# QA Analytics Platform

Self-hosted full-stack platform for collecting, analyzing, and visualizing automated test results.

## Goals

- Store test runs from pytest / Selenium / CI
- Show dashboards and historical trends
- Detect flaky tests
- Support AI agents for analysis and exploratory testing
- Demonstrate production-like architecture

## Tech Stack (planned)

- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Docker Compose
- Pytest
- Selenium
- GitHub Actions

Project initialization in progress.
